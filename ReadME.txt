Запуск проекта:

1. Запустить Visual Studio 2019 и склонировать проект с GitHub.
2. Запустить MS SQL Server Management Studio 18.8.
3. В окне "Имя сервера" ввести: (localdb)\MSSQLLocalDB, нажать Connect.
4. Создать новый запрос - нажать "New Query".
5. Скопировать содержимое файла script Poprijenok_BelyakovAA.sql.
6. Нажать выполнить. База данных создана и заполнена данными.
7. Запустить проект в Visual Studio.
